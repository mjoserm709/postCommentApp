# Backend

La documentacion principal del proyecto vive en el [`README.md`](../README.md) de la raiz.

Puntos relevantes del backend:

- NestJS + Mongoose
- JWT
- `ValidationPipe` global
- `GlobalExceptionFilter`
- `ResponseInterceptor`
- endpoints de `posts` y `comments`
- `POST /posts/bulk` con `insertMany()`
